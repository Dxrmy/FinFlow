**FinFlow** is a powerful terminal-based financial management system designed for power users. It moves beyond simple tracking, offering sophisticated debt-clearing strategies and multi-day income distribution logic to help you master your cash flow.

## Features
- 💸 **Smart Splitting**: Automatically distribute income based on Aggressive Debt Repayment or Balanced Saving strategies.
- 📈 **Bank Statement Analysis**: Process CSV and PDF statements to identify patterns and cost-reduction opportunities.
- 🛡️ **Debt Management**: Dedicated logic for tracking and prioritizing subscriptions, direct debits, and high-interest debts.
- ⌨️ **TUI Interface**: A lightning-fast terminal user interface for rapid data entry and analysis.
- 🗄️ **Persistent Storage**: Robust SQLite backend ensures your financial history stays local and secure.

## Tech Stack
- **Core**: Python 3.13
- **Database**: SQLite3
- **Analysis**: Custom PDF/CSV Parsing Engine
